# Security Policy

## Supported versions

Nemesis is at an early, pre-1.0 stage. Security fixes are applied to the latest
released version only.

| Version | Supported |
| --- | --- |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x: |

## Reporting a vulnerability

If you discover a security issue in Nemesis, please report it **privately**
rather than opening a public issue.

- Email: therealonslaughtgaming@gmail.com
- Subject line: `SECURITY: Nemesis`

Please include enough detail to reproduce the issue. You can expect an initial
acknowledgement within a few days. If the report is accepted, a fix will be
prioritized and you will be credited unless you prefer to remain anonymous; if
declined, you will receive an explanation.

Please do not disclose the issue publicly until a fix has been released.
